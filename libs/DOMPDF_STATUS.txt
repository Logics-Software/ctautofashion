1. Detect Dompdf installation
2. Generate PDF from Work Order data
3. Auto-download PDF file with format: WorkOrder_[NoOrder]_[Timestamp].pdf

1. Open Transaksi Work Order
2. Click detail of any Work Order (StatusOrder <= 2)
3. Click "Print Work Order PDF" button

================================================================================
  TROUBLESHOOTING
================================================================================

If PDF download fails:
1. Check error log in browser console
2. Verify libs/dompdf/autoload.inc.php exists
3. Check PHP memory limit (recommended: 128MB+)
4. Verify all dependencies are in libs/dompdf/lib/
